# Create contact

### Step 1: Open Add Contact Form
![Click Scan Code](/img/tutorial/getting-started/create-contact/create-contact-click-add-contact-button.png)
Go to Contacts page. Click on the add contact button.

### Step 2: Save Contact
![Click Scan Code](/img/tutorial/getting-started/create-contact/create-contact-save-contact.png)
Fill in the `First Name`, `Last Name`, and `Phone`. For `Phone` make sure it is a valid WhatsApp number. 

Click "Save". 

## Step 3: See this new contact in contacts page
![Click Scan Code](/img/tutorial/getting-started/create-contact/create-contact-contact-saved.png)
This is the contact details page. To go back to the contacts page from here, click the back icon (<-).

![Click Scan Code](/img/tutorial/getting-started/create-contact/create-contact-contact-saved-in-contacts-page.png)
You should see the contact in contacts page.
