# Add Trigger
In ExaBloom, a trigger is a rule-based automation that performs an action in response to a specific event or condition. 
It helps automate tasks and workflows, such as sending a message or assigning a task, saving time and improving efficiency.

Let's set up a trigger for our automation.

### Step 1: Find the trigger
![Start New Workflow](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-auto-responder/create-whatsapp-automation-auto-responder-initialise-adding-trigger.png)
Click "Add New Workflow Trigger" to open up the Workflow Trigger list.

![Start New Workflow](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-auto-responder/create-whatsapp-automation-auto-responder-search-for-trigger.png)
Search for a trigger named "Customer Replied".

![Start New Workflow](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-auto-responder/create-whatsapp-automation-auto-responder-select-customer-replied-trigger.png)
Select the "Customer Replied" trigger.

![Start New Workflow](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-auto-responder/create-whatsapp-automation-auto-responder-customer-replied-trigger.png)
This is the default state of the "Customer Replied" trigger.

### Step 2: Update the trigger
![Start New Workflow](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-auto-responder/create-whatsapp-automation-auto-responder-configuring-customer-replied-trigger.png)
Change the `Workflow Trigger Name` from "Customer Replied" to "Customer Replied (General Enquiry)".

Change the `Filters` to `Has Tag` and select a tag called `test-whatsapp`

### Step 3: Save the trigger
![Start New Workflow](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-auto-responder/create-whatsapp-automation-auto-responder-configured-customer-replied-trigger.png)
Click "Save Trigger" button to add the trigger to the workflow.
