# Verify WhatsApp Automation: Auto Responder Works
Now that you've completed your WhatsApp Automation: Auto Responder, it is time to test it out.

For the contact with tagged with `test-whatsapp`, get them to send a WhatsApp message to the WhatsApp that is connected to ExaBloom.

ExaBloom will automatically respond once a message is received from any contact that is tagged is `test-whatsapp`.
