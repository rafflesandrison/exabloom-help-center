# If Contact Reply "1" Branch
The workflow expects the contact to respond either 1 or 2 to the first message sent.

To handle this, we can add a "If / Else" action.

If contact respond "1", do this. <-- let's build this now
Else if contact respond "2", do this.

### Step 1: Add IF / ELSE Action
![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-1.png)
Find IF / ELSE action.

![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-2.png)
This is what IF / ELSE action look like.

### Step 2: Update action name
![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-3.png)
Change the action name to "Contact reply is?"

### Step 3: Update branch #1
![Update IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-4.png)
Change the name for the first branch to "Replied 1".

Under segments, choose "Contact Reply"...

![Update IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-5.png)
... then choose "Replied message".

![Update IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-6.png)
For the operator, choose "Is".

![Update IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-7.png)
For the value, type "1". 

This is because in this branch is only active when the contact reply with a message "1".

### Step 4: Save action
![Save IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-8.png)
Save the IF / ELSE action by clicking on "Save Action" button.

![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-9.png)
Branch #1 ("Replied 1") is added to your workflow.
