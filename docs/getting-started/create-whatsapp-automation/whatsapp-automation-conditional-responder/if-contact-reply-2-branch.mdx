# If Contact Reply "2" Branch

The workflow expects the contact to respond either 1 or 2 to the first message sent.

To handle this, we can add a "If / Else" action.

If contact respond "1", do this.
Else if contact respond "2", do this. <-- let's build this now

### Step 1: Edit IF Else Action
![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-10.png)
Click on the IF Else Action "Contact reply is?" in the workflow to start editing the action.

### Step 2: Add 2nd branch
![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-11.png)
Click "Add Branch" to add a branch to the workflow.

![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-12.png)
Now you have the 2nd branch...

![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-13.png)
In this 2nd branch, update the following...

Change the 2nd branch name to "Replied 2".

Change the 2nd branch segment condition to "Replied message" is "2".

Click "Save Action" to add the 2nd branch to the workflow.

### Step 3: Save workflow
![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-14.png)
Click "Save" to save the workflow.

![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-if-else-action-15.png)
Workflow is now saved !

Congratulations! You have successfully added the 2nd branch that is activated when the contact replied "2". 👍🏼
