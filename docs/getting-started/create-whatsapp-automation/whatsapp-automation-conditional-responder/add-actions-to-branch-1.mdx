# Add actions to "Replied 1" branch

### Step 1: Open Actions list
![Add action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-add-action-to-branch-replied-1-1.png)
Click on the plus icon to start adding an action under "Replied 1" branch.

### Step 2: Add Send SMS Action
![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-add-action-to-branch-replied-1-2.png)
Add Send SMS Action.

![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-add-action-to-branch-replied-1-3.png)
This is Send SMS action default state.

![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-add-action-to-branch-replied-1-4.png)
Change the following...

Change action name to "WhatsApp: Send pricing info".

Change message to the following

    Here's the pricing info:

    Fried chicken: $15 per bucket
    Fries: $2 (S), $4 (M), $4.50 (L)


Click "Save Action" to add the Send SMS action. 

This means that once the contact enter the "Replied 1" 
branch, the contact will receive the message from action "WhatsApp: Send pricing info".

![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-add-action-to-branch-replied-1-5.png)
Click "Save" to save the workflow.


![Find IF ELSE action](/img/tutorial/getting-started/create-whatsapp-automation/whatsapp-automation-conditional-responder/conditional-responder-add-action-to-branch-replied-1-6.png)
Workflow is saved !
